Belly Button Biodiversity Dashboard


Use the interactive charts below to explore the dataset

Our dataset contains the results of an experiment involving 150 participants. Some tests were conducted on the swab samples from each individual's belly button to get the information about the existing bacteria species in that area . You can select a test subject ID from the dropdown menu to see the test result in the provided charts on this page. Also, there is a "Demographic Info" panel that gives some information about the selected test subject.
